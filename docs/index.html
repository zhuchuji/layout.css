Grid system

Features

The grid system is mobile-first, which implement with flexbox. "row" and "col" are two basic components in the grid system, where there are 24 columns in a row. The grid system could help you implement various layouts, but keep in mind that "mobile first" means you are developing websites on mobile devices by default.

There are a few useful classes for alignment.

  • .row--vertical: change the direction
  • .row--no-wrap: unwrap the column when overflowing
  • .row--justify-center: justify the content at the center of the row
  • .row--justify-end: justify the content at the end of the row
  • .row--justify-space-between: justify the content with space-between
  • .row--justify-space-evenly: justify the content with space-evenly

Responsive system. Below are media breakpoints:

  • default: mobile device
  • sm -- min-width: 768px
  • md -- min-width: 992px
  • lg -- min-width: 1200px
  • xl -- min-width: 1900px
