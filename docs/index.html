Grid system

Features

The grid system is a mobile-first flexbox grid for layout. "row" and "col" are basic components, where there are 24 columns in a row. The grid system could help you implement various layouts.

Responsive layout. Media queries are:

  • xs -- max-width: 767px
  • sm -- min-width: 768px
  • md -- min-width: 992px
  • lg -- min-width: 1200px
  • xl -- min-width: 1900px
col-xs-24 col-sm-20 col-md-16 col-lg-12
col-sm-20 col-sm-offset-4 col-md-16 col-md-offset-8 col-lg-12 col-lg-offset-12

If you need margin between columns, you could set padding in columns

col-8 gutter:10%
col-8 gutter: 10%
