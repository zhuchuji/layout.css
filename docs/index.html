Grid system

Overview

The grid is a mobile-first, 24-column and fully responsive grid system. We provide wrapping containers, grids, responsive utility and typical layouts with a lot of predefined classes, which could help you implement various kinds of layouts, but keep in mind that "mobile first" means you are developing websites on mobile devices by default.

Containers

We provide fixed-width and fluid-width containers. Fixed-width container will change its max-width according to media breakpoints. Fluid-width container-fluid will keep 100% wide on all devices. We add 15px padding on containers, which could be overwrited by custom classes. Even sometimes you may find out you don't need a container since row takes up 100% wide of its parent.

row and col are the basic components in the grid system. And col must be the direct child of row.

col are used for equal-width distributions with flexbox's automatic caculation for layout. In the above example, col will make the column equal on size on all devices. The columns will automatically adjust their size equally when adding or removing column into the row as well as you resize the window.

Of cousrse you could use col-*, and col-*-* for your layout if the layout is known and unchanged.

col-offset-* and col-offset-*-* are for column offset to the left. When using col-offset-* and col-offset-*-*, you must not set margin in CSS, since they are implemented with margin and will be overwritted.

col-push-*, col-pull-*, col-push-*-* and col-pull-*-* are used to set your column's position relative to its original default position.

Gutter

We don't provide gutters between columns, since it might be any value in real websites. For present, we could implement similar effect with flex justfiy-content: space-between, but it's not a fixed-width gutter. We will implement this function with CSS Grid Layout in future. If you need gutters between columns, you could set margin or padding in columns.

col gutter: 10%
col gutter: 10%
col gutter: 10%
<div class="row basic-row">
  <div class="col block--gutter block--first">
    <div class="block--blue">col gutter: 10%</div>
  </div>
  <div class="col block--gutter">
    <div class="block--green">col gutter: 10%</div>
  </div>
  <div class="col block--gutter block--last">
    <div class="block--blue">col gutter: 10%</div>
  </div>
</div>
<style>
  .block--gutter {
    margin: 0 5%;
  }
  .block--first {
    margin-left: 0;
  }
  .block--last {
    margin-right: 0;
  }
</style>

To add gutter between columns, you should add customed classes with margin or padding on column rather than modifying directly on predefined classes col-*. Otherwise, it will affect all columns on your website and which completely is not what you want.

Responsive column

We build the responsive grids with media query, so that it could meet the need for responsive web. Media breakpoints are set according to WebDimensions, which statisic most users' devices worldwide.

col and col-* are used for mobile layout by default. col-sm and col-sm-* are used for small device layout. col-md and col-md-* are used for medium device layout. col-lg and col-lg-* are used for large device layout and col-xl and col-xl-* are used for extra large device layout. Besides, col-offset-*, col-offset-sm-*, col-offset-md-*, col-offset-lg-*, and col-offset-xl-* are provided as well as col-push-*, col-pull-*, col

col-24 col-sm-20 col-md-16 col-lg-12 col-xl-8
<div class="col-24 col-sm-20 col-md-16 col-lg-12 col-xl-8">
  <div class="block--blue">
    col-24 col-sm-20 col-md-16 col-lg-12 col-xl-8
  </div>
</div>

Responsive columns are designed from small devices and up, e.g, col-24 occupy 24 grids on extra small devices and up, col-sm-20 will overwrite the width on small devices and up and so on. Combining different responsive column classes is common and effective in responsive web.

Justify And Alignment

We provide useful classes for justify and alignment.

  • row--vertical: change the direction to vertical
  • row--no-wrap: unwrap the column when overflowing
  • row--justify-center: justify the content at the center of the row
  • row--justify-end: justify the content at the end of the row
  • row--justify-space-between: justify the content with space-between
  • row--justify-space-evenly: justify the content with space-evenly
  • row--align-start: align the content at flex start
  • row--align-center: align content at the center
  • row--align-end: align content at the end
